## 🔥 TIER A TOPIC BREAKDOWN (Must Master)

### **Unit I (Project Planning)**

| Topic                      | Frequency | Focus Areas                                                | PYQ Pattern                       |
| -------------------------- | --------- | ---------------------------------------------------------- | --------------------------------- |
| SPM Definition/Importance  | 4/4       | Activities, methodologies, principles, real-world examples | Sec A (2 marks) + Sec B (7 marks) |
| Categorization of Projects | 3/4       | Types: compulsory, strategic, operational; examples        | Sec A + Sec C                     |
| Cost-benefit Evaluation    | 3/4       | **NPV, ROI, Payback Period calculations** (numerical)      | Sec C (numerical question)        |

**Key Numerical Formula:**

- **NPV** = Σ [Cash Flow / (1+r)^n] - Initial Investment
- **ROI** = (Total Revenue - Total Cost) / Total Cost × 100
- **Payback Period** = Years to recover initial investment

### **Unit II (Effort Estimation)**

| Topic                    | Frequency | Focus Areas                                            | PYQ Pattern               |
| ------------------------ | --------- | ------------------------------------------------------ | ------------------------- |
| RAD Model                | 4/4       | Features, phases, advantages, when to use              | Sec B + Sec C             |
| Agile Methodologies      | 4/4       | Principles, comparison with traditional, Scrum/XP      | Sec C                     |
| Function Points (COSMIC) | 3/4       | **FP calculation** (numerical), components, weightages | Sec B + Sec C (numerical) |
| Project Life Cycle       | 3/4       | Stages, activities per stage                           | Sec A (definition)        |

**FP Calculation Formula:**

- Unadjusted FP = (EI × weight) + (EO × weight) + (EQ × weight) + (ILF × weight) + (EIF × weight)
- Adjusted FP = UFP × [0.65 + (0.01 × TDI)]

### **Unit III (Activity Planning & Risk)**

| Topic                        | Frequency | Focus Areas                                               | PYQ Pattern                    |
| ---------------------------- | --------- | --------------------------------------------------------- | ------------------------------ |
| Critical Path Method (CPM)   | 4/4       | **Network diagram, Forward/Backward pass, Critical Path** | Sec C (diagram + calculations) |
| Forward Pass & Backward Pass | 3/4       | EST, EFT, LST, LFT calculations                           | Sec C                          |
| PERT Technique               | 3/4       | Three-time estimates, expected time, variance             | Sec B + Sec C                  |
| Risk Management              | 4/4       | Types, identification, assessment, mitigation             | All sections                   |

**CPM/PERT Key Formulas:**

- **EST** = Maximum of EFT of predecessors
- **EFT** = EST + Duration
- **LST** = LFT - Duration
- **LFT** = Minimum of LST of successors
- **PERT Expected Time** = (Optimistic + 4×Most Likely + Pessimistic) / 6

### **Unit IV (Management & Control)**

| Topic                    | Frequency | Focus Areas                                               | PYQ Pattern       |
| ------------------------ | --------- | --------------------------------------------------------- | ----------------- |
| Earned Value Analysis    | 4/4       | **EV, PV, AC, SPI, CPI, EAC, VAC** calculations           | Sec C (numerical) |
| Framework for Management | 3/4       | Collection of data, visualizing progress, control actions | Sec B + Sec C     |

**EVA Key Formulas:**

- **CPI** = EV / AC (Cost Performance Index)
- **SPI** = EV / PV (Schedule Performance Index)
- **EAC** = BAC / CPI (if only CPI influences)
- **EAC** = AC + [(BAC - EV) / (CPI × SPI)] (if both influence)
- **VAC** = BAC - EAC

## 🎓 SECTION-WISE STRATEGY

### **Section A (7 × 2 = 14 marks)**

**Strategy:** Must score 12+/14

- **Length:** 3-5 lines per answer (half page)
- **Time:** 2-3 minutes per question (20 minutes total)
- **Focus:** Definitions, key points, brief explanations
- **Common Topics:** SPM definition, Project types, Agile definition, Risk types, CPM definition, EVA definition, Organizational behavior

**Writing Tips:**

- Start with a 1-line definition
- Follow with 2-3 key points
- Add a relevant example if space permits
- No diagrams needed unless asked

### **Section B (3 × 7 = 21 marks)**

**Strategy:** Choose wisely, score 18+/21

- **Length:** 1.5-2 pages per answer
- **Time:** 20-25 minutes per question (75 minutes total)
- **Focus:** Detailed explanations with structure
- **Question Types:**
  - Explain/Discuss topics (RAD, PERT, SCM)
  - Calculate (Function Points, SPI/CPI - easier ones)
  - Compare/Contrast

**Writing Tips:**

- Start with introduction (2-3 lines)
- Use numbered points for processes/steps
- Include a simple diagram if it helps (optional)
- Conclude with significance/application
- For "Discuss" - cover 4-5 aspects in detail
- For "Explain" - go deep into mechanism/working

### **Section C (5 × 7 = 35 marks)**

**Strategy:** Do 4 very well, 1 decent = 28+/35

- **Length:** 1.5-2 pages per answer
- **Time:** 25 minutes per question (125 minutes total)
- **Focus:** In-depth answers with examples/numericals
- **Guaranteed Topics:**
  - CPM/PERT network (Unit III) - **ALWAYS APPEARS**
  - EVA calculations (Unit IV) - **ALWAYS APPEARS**
  - RAD or Agile (Unit II) - **ALWAYS APPEARS**
  - NPV/ROI (Unit I) or Oldham-Hackman (Unit V)

**Writing Tips:**

- For numerical problems:
  - Show all formulas first
  - Show step-by-step calculations
  - Box the final answer
  - Write units (weeks, rupees, etc.)
- For theoretical questions:
  - Introduction → Main body (with headings) → Conclusion
  - Use comparison tables where applicable
  - Diagrams are highly valued in Section C
  - Real-world examples add marks

## ⚡ QUICK EXAM TIPS

### **Before the Exam:**

1. **Must revise the night before:** All formulas (NPV, CPM, EVA), all diagrams
2. **Sleep 7+ hours** - exhaustion kills calculation accuracy
3. **Carry:** Calculator (for NPV/EVA), extra pens, admit card

### **During the Exam:**

1. **First 10 minutes:** Read entire paper, mark the questions you'll attempt
2. **Time allocation:**
   - Section A: 20 minutes
   - Section B: 75 minutes (25 min × 3)
   - Section C: 125 minutes (25 min × 5)
   - Buffer: 10 minutes
3. **Attempt order:** Section B → Section C → Section A (save easiest for last when tired)
4. **For calculations:** Double-check formulas before computing
5. **If stuck:** Move on, come back later

### **Common Mistakes to Avoid:**

1. ❌ Confusing CPI and SPI formulas (learn: **C** for **C**ost, **S** for **S**chedule)
2. ❌ Missing units in numerical answers (₹, weeks, person-months)
3. ❌ Drawing CPM/PERT diagrams without labels
4. ❌ Not showing calculation steps (you get partial marks for method)
5. ❌ Writing vague answers without structure
6. ❌ Spending too much time on one question
